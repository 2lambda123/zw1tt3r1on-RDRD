# Recursive Docker Registry Downloader

I created this script to be able to recursively download docker registry files and to be able to easily investigate them.

# Python 3 Requirements

```
pyfiglet==1.0.2
Requests==2.31.0
wget==3.2
```

```
python3 rdrd.py
```
